# Development usage
install docker (https://docs.docker.com/get-docker/).

## Clone the repo
```bash
git clone https://github.com/adeliiman/FastAPI-Copy-Trade.git
```

  
## Build 
```bash
docker-compose up -d --build
```

Now visit <http://localhost:8000/docs>
